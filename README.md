# Algorithms & Data Structures

Сборник задач по алгоритмам и структурам данных на C++. Каждая папка — отдельная тема, внутри 6 задач нарастающей сложности (A→F).

## Содержание

| Раздел | Тема |
|--------|------|
| `containers/` | Стек, очередь, применения |
| `dynamic_programming/` | Динамическое программирование |
| `graphs_dfs/` | Графы — обход в глубину (DFS) |
| `graphs_bfs_dijkstra/` | Графы — BFS и алгоритм Дейкстры |
| `segment_tree/` | Дерево отрезков |
| `avl_tree/` | АВЛ-дерево |

## Разделы

### Контейнеры (`containers/`)
- `task_a_brackets` — проверка правильности скобочной последовательности
- `task_b_brackets_multi` — проверка последовательности из нескольких типов скобок
- `task_c_postfix` — вычисление постфиксного выражения
- `task_d_stack_min` — стек с операцией получения минимума за O(1)
- `task_e_queue_min` — очередь с операцией получения минимума за O(1)
- `task_f_balls` — задача с шарами на стеке/очереди

### Динамическое программирование (`dynamic_programming/`)
- `task_a_fibonacci` — числа Фибоначчи
- `task_b_paid_staircase` — платная лесенка (минимальная стоимость подъёма)
- `task_c_max_cost_route` — маршрут максимальной стоимости на сетке
- `task_d_knight_move` — ход конём на доске (подсчёт путей)
- `task_e_knight_move_2` — ход конём, усложнённый вариант
- `task_f_calculator` — калькулятор через ДП

### Графы — DFS (`graphs_dfs/`)
- `task_a_edge_list_to_adjacency_matrix` — преобразование списка рёбер в матрицу смежности
- `task_b_vertex_degrees` — степени вершин по списку рёбер
- `task_c_graph_traversal` — обход графа в глубину
- `task_d_cheat_detection` — обнаружение списывания через компоненты связности
- `task_e_cycle_detection` — определение наличия цикла в графе
- `task_f_tree` — проверка, является ли граф деревом

### Графы — BFS и Дейкстра (`graphs_bfs_dijkstra/`)
- `task_a_bfs` — обход графа в ширину
- `task_b_evacuation` — эвакуация (кратчайший путь через BFS)
- `task_c_01_bfs` — 0-1 BFS (рёбра с весами 0 и 1)
- `task_d_pythagorean_express` — задача с Дейкстрой на взвешенном графе
- `task_e_dijkstra_path` — кратчайший путь алгоритмом Дейкстры с восстановлением
- `task_f_numbers` — задача на поиск в графе состояний

### Дерево отрезков (`segment_tree/`)
- `task_a_segment_sum` — сумма на отрезке
- `task_b_rmq` — минимум на отрезке (Range Minimum Query)
- `task_c_gcd_segment` — НОД на отрезке
- `task_d_count_max` — подсчёт максимальных элементов на отрезке
- `task_e_distinct_count` — количество различных элементов на отрезке
- `task_f_kth_zero` — поиск k-го нуля

### АВЛ-дерево (`avl_tree/`)
- `avl_tree` — реализация самобалансирующегося бинарного дерева поиска

## Стек

- **Язык:** C++17
- **IDE:** Visual Studio 2022
- **Компилятор:** MSVC

## Сборка

Каждый файл — самостоятельная программа. Чтобы запустить в Visual Studio:

1. **Create a new project** → **Empty Project (C++)**
2. В Solution Explorer: правый клик на **Source Files** → **Add → Existing Item** → выбрать `.cpp`
3. **Ctrl+F5** — запуск без отладки

В одном проекте одновременно только один `.cpp` (у каждого своя функция `main`).
